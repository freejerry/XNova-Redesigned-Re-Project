Setup Complete
You need to create an admin account.
When you press finish, you will be taken to the login page, which will be shown to everyone, simply register an account,
The first account that is registered will be given administrative privileges,
you can then use this account to grant administrative privilages to other users from the admin panel.

Press next to proceed to the login screen.

It is highly recomended that you now delete the install directory so that no-one else can re-configure your server.
